Syntax - Picture mediatype

The picture media type allows you to show pictures within your ximpel presentation.

playlist.xml

						<ximpel>
							<playlist>
							    <subject>
							        <media>
										<picture src="demo.png" />
							        </media>
							    </subject>
							</playlist>
						</ximpel>

The src attribute specifies the filename of the picture that is to be played.


Positioning of the picture

The x, y, width and height attributes can be used to position the picture at a certain spot within the presentation and force it to have certain dimensions. If you don't set any of these attributes then the picture expands as large as possible such that it covers up the entire width and/or height while maintaining aspect ratio. 

						<picture src="demo.png" x="200px" y="200px" width="400px" height="400px" />

By setting the x and/or y attributes, the picture will have an offset from the top-left of the view area. Using the x and y offsets do not change the default width and height used for the picture (ie. the width/height is such that it expands to fill the view-area while maintaining the aspect ratio). This may cause the picture to fall partly outside the view-area. Usually you want to provide width/height attributes when using x and y offsets. By specifying the width and/or height attributes you can change the width and height that the picture will have.


Playing the picture for a limited time

By default, the picture is played indefinitely and would require some user interaction (clicking an overlay for example) to jump to a different part of the presentation. However, the duration attribute is available for every media type and can be used to limit the play time of the picture. It specifies the number of seconds the picture should play.

This picture will play for 5 seconds

<picture src="demo.png" duration="5" />
